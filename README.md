Shopping project!

This program is supposed to emualate a basic shopping sequence. 

Features:
- 5 python files housing various functions + 2 json files featuring data dictionaries
- Creation of user account and cart states
- Ability to add and remove items from the cart
- Ability to purchase lifetime subscriptions, which allow discounts
- Tax
- Dynamic menu generation that autonomically updates based on dictionary data (minimal hard-coding unless it's regarding choices that wont be updated like yes/no)

Deferred features:
- OOP inclusion: Moved for the next project that'll have object design from the start
- Monthly subscriptions: Moved for a proper backend project in a few months due to data/time tracking paired with consistent user state
- User accounts: Skipped in favor of starting another, more advanced program
